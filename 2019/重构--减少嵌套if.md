### 2.重构，用 OO 的继承或者组合(结合设计模式)
~~~
例子：
1.如果是qq，则qq登录
2.如果是wx，则wx登录
3.如果是phone，则phone登录
~~~
解决：
定义一个接口 loginer
~~~
type interface loginer{
    Login(ctx context.context)error
}
~~~
qq、wx、phone 分别在各自子类中去实现Login(),实质 设计模式中的模板方法
